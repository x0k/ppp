Требования к реализации операции `payment`:

- PayPal - увеличивает `base` на `amount`
- WebMoney - увеличивает `base` на `2*amount`
- BankCat - увеличивает `base` в `amount` раз

> Для тестовых данных результат операций будет целочисленным положительным числом.
