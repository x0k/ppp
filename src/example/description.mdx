# Платежные системы

Необходимо выполнить интеграцию с тремя платежными системами, реализовав следующие операции:
`payment` и `payout`.

Требования к реализации операции `payment`:

- PayPal
  - Уменьшает `amount` на `1`
- Webmoney
  - Уменьшает `amount` на `2`
- BankCat
  - Увеличивает `amount` на `3`

Требования к реализации операции `payout`:

- Использует логику расчета `payment` и уменьшает итоговое значение на `4`
