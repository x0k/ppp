# Платежные системы

Необходимо выполнить интеграцию с тремя платежными системами, реализовав следующие операции:
`payment` и `payout`.

Требования к реализации операции `payment`:

- PayPal - увеличивает `amount` на `1`
- WebMoney - увеличивает `amount` на `2`
- BankCat - увеличивает `amount` в `2` раза

Требования к реализации операции `payout`:

- Использует логику расчета `payment` и уменьшает итоговое значение на `1`
